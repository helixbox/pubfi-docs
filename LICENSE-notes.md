# License Notes

This repository was migrated from a public-safe documentation snapshot in `pubfi-mono`.

- source snapshot license context: `GPL-3.0`
- snapshot commit: `38852a16c5af`

The public docs repository now uses a split license:

- documentation text, diagrams, and non-code assets: `CC-BY-4.0`
- code examples, scripts, configuration, and automation files: `MIT`

Rationale:

- `CC-BY-4.0` is better suited than GPL for public docs because it allows quoting, reuse, and
  redistribution with attribution.
- `MIT` keeps runnable examples easy to copy into agent clients, demos, and integration tests.
- GPL remains acceptable for software repositories, but it is unnecessarily heavy for public docs
  whose goal is discoverability, citation, and reuse.

See [LICENSE.md](LICENSE.md), [LICENSE-DOCS.md](LICENSE-DOCS.md), and
[LICENSE-CODE.md](LICENSE-CODE.md).
