# Assets

This directory is reserved for public-safe diagrams, screenshots, and images after the docs move to
a public repository.

Allowed assets:

- architecture diagrams that do not show secrets or private deployment details;
- public product screenshots that contain no accounts, keys, usage, billing, or customer data;
- source-selection diagrams;
- sanitized example response diagrams;
- logos or icons that are approved for public use.

Do not add:

- private dashboard screenshots;
- production database screenshots;
- account or billing screenshots;
- raw answer-engine outputs;
- unredacted crawler logs;
- credentials, wallet secrets, or API keys.

Every asset should have a short source note in the page that uses it.
